# India Census SQL Analysis

> "A comprehensive SQL-based analysis of the 2011 Indian Census data, focusing on state-level literacy rates, sex ratios, and population growth trends using advanced aggregation and ranking techniques."

---

## 📊 Project Overview (STAR Method)

### **Situation**
The goal was to perform a comprehensive data analysis on the **2011 Indian Census Data** to extract meaningful demographic insights. This information is critical for government planning and identifying regional disparities.

### **Task**
My objective was to import raw data from Excel into **Microsoft SQL Server** and write complex SQL queries to calculate key metrics such as average growth rates, sex ratios, and literacy rates.

### **Action**
I performed the following technical steps using advanced SQL techniques:
* **Data Integration:** Imported two primary datasets into SQL Server.
* **Aggregate Analysis:** Used `GROUP BY`, `SUM`, and `AVG` functions for state-level demographics.
* **Filtering & Ranking:** Implemented `HAVING` clauses and `TOP` functions to rank performing regions.
* **Advanced Scripting:** Utilized **Temporary Tables** and **Union Operators** for unified reporting.
* **Pattern Matching:** Employed `LIKE` operators for specific state filtering.

### **Result**
The project successfully generated a detailed demographic profile of India, identifying the top 3 states with the highest growth rates and providing a data-driven foundation for policy recommendations.
